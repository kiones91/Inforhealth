# Inventário de Assets — Portal Inforhealth

Este documento contém o inventário completo dos ativos de mídia (imagens, logotipos e ícones) do novo portal Inforhealth Educação.

---

## 1. Estrutura de Diretórios

Os assets estão organizados sob a pasta raiz `assets/imagens/` nas seguintes subpastas:

```
assets/imagens/
├── institucional/       # Identidade visual, logos e favicon
├── equipe/              # Fotos dos docentes e equipe da empresa
├── cursos/              # Imagens de destaque de cada curso (36 imagens)
├── hospitais/           # Logotipos de hospitais e operadoras atendidas
└── parceiros/           # Logotipos de parceiros comerciais (vazio)
```
